Q1: WHAT IS THIS VIDEO ABOUT?
-----------------------------
A man (Sergey Dashevsky, Russia) holds the Guinness World Record for the
world's smallest rideable bicycle at just 8.4 centimeters (3.31 inches).
The video reveals he built progressively smaller bikes since 2008, and that
an unofficial builder has made one at just 5.3 cm — not yet Guinness verified.
The reveal arc: tiny bike → scale comparison → real Moscow traffic riding.

Q3: HOW CAN THIS BE REUSED FOR ANOTHER TOPIC?
----------------------------------------------
The architecture is fully reusable:

  production/output/{topic-name}/
  ├── script.txt       ← replace with new 45s script
  ├── captions.json    ← replace with new timed captions
  ├── cutlist.json     ← replace with new scene timings
  ├── capcut_project.json ← CapCut edit template
  ├── public/audio/
  │   └── voiceover.mp3 ← generate new via FreeTTS API
  └── out/
      └── preview.mp4  ← render with new script

Workflow for new topic:
  1. Run WebSearch for topic facts
  2. Write script.txt (ClpQuips HOOK/REVEAL/STING/OUTRO)
  3. Update captions.json timestamps
  4. Update cutlist.json scene timings
  5. Generate TTS: POST to https://freetts.org/api/tts
  6. Update gen_video_v2.py captions list
  7. Run: python3 gen_video_v2.py
