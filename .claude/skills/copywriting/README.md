## What This Is

Expert conversion copywriting for marketing pages — homepage, landing pages, pricing pages, feature pages, about pages, and product pages. Covers headline formulas, CTA copy, benefits vs. features, and page structure frameworks.

## Quick Start

Before writing, gather:
1. **Page purpose** — one primary action
2. **Audience** — ideal customer, problem, objections, language they use
3. **Product/offer** — what makes it different, key transformation
4. **Traffic source** — what do visitors already know?

## Core Principles

| Principle | Bad | Good |
|-----------|-----|------|
| Clarity over cleverness | "Leverage synergies to optimize value" | "Cut your reporting from 4 hours to 15 minutes" |
| Benefits over features | "GPS tracking included" | "Never lose a package again" |
| Specificity over vagueness | "Saves time" | "Saves 3 hours per week" |
| Active over passive | "Your order is shipped by us" | "We ship your order today" |
| Customer language | "Deploy containerized microservices" | "Put your code on the internet" |

## Page Structure

```
Above the fold:
  Headline (value prop) → Subheadline → Primary CTA

Sections:
  Social proof → Problem/Pain → Solution/Benefits → How It Works → FAQ → Final CTA
```

## Headline Formulas

- `{Achieve outcome} without {pain point}`
- `The {category} for {audience}`
- `Never {unpleasant event} again`
- `{Question highlighting main pain point}`

## CTA Guidelines

**Weak:** Submit, Sign Up, Learn More, Click Here
**Strong:** Start Free Trial, Get the Complete Checklist, See Pricing for My Team
